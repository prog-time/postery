# Подключение AI

Postery умеет автоматически улучшать тексты постов с помощью AI. Поддерживается два провайдера: **OpenAI** и **GigaChat**.

> Одновременно может быть активен только один провайдер.

---

## OpenAI (ChatGPT)

### Что нужно
- API-ключ OpenAI

### Как получить ключ
1. Зарегистрируйтесь на [platform.openai.com](https://platform.openai.com)
2. Перейдите в раздел **API Keys**
3. Нажмите **Create new secret key**
4. Скопируйте ключ — он показывается только один раз

### Подключение
1. Откройте **AI Провайдеры → Добавить**
2. Выберите тип: `OpenAI`
3. Вставьте API-ключ
4. При необходимости укажите кастомный Base URL (если используете прокси)
5. Поставьте галочку **Активен**
6. Сохраните

---

## GigaChat (Сбер)

### Что нужно
- Client ID и Client Secret от GigaChat API

### Как получить доступ
1. Зарегистрируйтесь на [developers.sber.ru](https://developers.sber.ru)
2. Создайте проект и подключите GigaChat API
3. Получите Client ID и Client Secret

### Подключение
1. Откройте **AI Провайдеры → Добавить**
2. Выберите тип: `GigaChat`
3. Заполните Client ID и Client Secret
4. Выберите scope: `GIGACHAT_API_PERS` (личный) или `GIGACHAT_API_CORP` (корпоративный)
5. Поставьте галочку **Активен**
6. Сохраните

## Использование AI при создании поста

На шаге 3 визарда создания поста рядом с полями **Заголовок** и **Текст** появляется кнопка **✨ AI**. При нажатии AI переформулирует текст на основе промпта, настроенного для данного канала.

### Настройка промптов

Для каждого канала можно задать отдельные промпты:

1. Откройте **Источники** → выберите нужный канал → **Редактировать**
2. Перейдите на вкладку **AI Prompts**
3. Заполните:
   - **Промпт для заголовка** — инструкция для AI по переформулировке заголовка
   - **Промпт для текста** — инструкция для AI по переформулировке основного текста
4. Сохраните
