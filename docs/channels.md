# Подключение каналов

Перед публикацией постов нужно подключить хотя бы один канал. Все каналы настраиваются в разделе **Источники** в админ-панели.

---

## Telegram

### Что нужно
- Токен Telegram-бота
- ID канала или группы

### Как получить токен бота
1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям — придумайте имя и username для бота
4. BotFather выдаст токен вида `123456789:AABBccDDee...` — скопируйте его

### Как получить ID канала
- Для публичного канала: `@имя_канала` (например, `@mychannel`)
- Для приватного: добавьте бота в канал, затем перешлите любое сообщение из канала боту [@userinfobot](https://t.me/userinfobot) — он покажет ID

### Подключение
1. Откройте **Источники → Telegram → Добавить**
2. Заполните название, токен бота и ID канала
3. Нажмите **Проверить** — система отправит тестовое сообщение
4. Сохраните

## ВКонтакте

### Что нужно
- Токен доступа группы ВКонтакте
- ID группы

### Как получить токен
1. Откройте управление группой → **Настройки → Работа с API**
2. Создайте ключ доступа с правами: `wall`, `photos`
3. Скопируйте токен

### Как получить ID группы
ID отображается в адресе группы. Для групп с числовым URL — это и есть ID. Для групп с адресом вида `vk.com/mygroup` воспользуйтесь сервисом [vk.com/dev](https://vk.com/dev).

### Подключение
1. Откройте **Источники → ВКонтакте → Добавить**
2. Заполните название, токен и ID группы
3. Нажмите **Проверить**
4. Сохраните
