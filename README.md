# Postery

Postery — это сервис для публикации постов сразу в несколько соцсетей и мессенджеров. Написал один раз — опубликовал везде.

## Что умеет

- **Публикует в Telegram, ВКонтакте и MAX** — одну публикацию сразу во все нужные каналы
- **По расписанию** — укажи дату и время, Postery опубликует сам
- **С картинками** — можно прикрепить одно или несколько фото
- **AI помогает с текстом** — если нужно переформулировать или улучшить текст, AI сделает это за вас
- **Разные тексты для разных площадок** — можно настроить отдельный текст для каждого канала
- **Календарь** — наглядно видно, что и когда запланировано

## Как запустить

1. Скачайте проект на свой компьютер или сервер
2. Запустите скрипт:

```bash
./start.sh
```

3. Откройте браузер и перейдите по адресу: **http://localhost:8000/admin**
4. Войдите: логин `admin`, пароль `admin`

> Перед использованием смените пароль через раздел «Пользователи».

## Как опубликовать пост

1. Перейдите в раздел **Публикации** → **Создать публикацию**
2. **Шаг 1** — напишите заголовок, текст и добавьте фото
3. **Шаг 2** — выберите, в какие каналы публиковать
4. **Шаг 3** — при необходимости настройте текст отдельно для каждого канала и укажите время публикации
5. Нажмите **Опубликовать**

## Подключение каналов

Перед публикацией нужно подключить хотя бы один канал:

- **Telegram** — нужен токен бота и ID канала
- **ВКонтакте** — нужен токен группы
- **MAX** — нужен токен и ID чата

Всё это настраивается в разделе **Источники** в админ-панели.

## Подключение AI

Посты можно улучшать с помощью AI. Поддерживается:
- **OpenAI** (ChatGPT)
- **GigaChat** (Сбер)

Ключ API добавляется в разделе **AI Провайдеры**. Одновременно может быть активен только один провайдер.

## Пользователи и роли

В системе два типа пользователей:

| Роль | Что может делать |
|------|-----------------|
| **Суперадмин** | Всё: посты, источники, пользователи, AI |
| **Редактор** | Только посты и источники |

Добавить или сменить суперадмина:

```bash
python create_superadmin.py
```

## Важно перед запуском в продакшен

### 1. Создайте файл `.env`

Скопируйте шаблон и задайте секретный ключ:

```bash
cp .env.example .env
```

### 2. Сгенерируйте секретный ключ

```bash
python -c "import secrets; print(secrets.token_hex(32))"
```

Вставьте результат в `.env`:

```
SECRET_KEY=<сгенерированный-ключ>
```

Приложение **не запустится**, если `SECRET_KEY` не задан.

> **Важно:** если вы смените `SECRET_KEY` после первого запуска, все зашифрованные значения
> в базе данных (токены ботов, API-ключи) станут нечитаемыми. Ротируйте ключ только
> с заранее подготовленным планом миграции данных.

### 3. Смените пароль администратора

```bash
python create_superadmin.py
```
