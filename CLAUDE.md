## Роли и агенты

Любая задача — фича, баг, рефакторинг, вопрос об архитектуре — **обязательно** проходит через агента `team-lead`.

**Правило:** Никогда не приступай к реализации напрямую. Сначала вызови `team-lead`.

Когда вызывать `team-lead`:
- Новая функциональность или изменение поведения
- Исправление бага
- Рефакторинг или изменение архитектуры
- Вопрос об устройстве кодовой базы
- Изменение схемы базы данных
- Любое другое изменение в проекте

Вызов агента:
```
используй Skill tool с параметром skill: "team-lead"
```

## Стек и ключевые файлы

**Технологический стек:**
- FastAPI + Uvicorn (Python 3.11+)
- Starlette-Admin (admin UI с `CustomView` и `ModelView`)
- SQLite + SQLAlchemy ORM (без Alembic)
- Jinja2 (шаблоны)
- Fernet (шифрование токенов), bcrypt (пароли)
- OpenAI gpt-4o-mini + GigaChat (Sber OAuth)

**Ключевые точки входа:**

| Файл/Директория | Назначение |
|-----------------|------------|
| `main.py` | Точка запуска, `_migrate()` — ручные миграции |
| `app/admin.py` | Регистрация всех admin-вьюх |
| `app/worker.py` | Фоновый воркер (опрос каждые 30 с) |
| `admin/templates/` | Jinja2-шаблоны для admin UI |
| `app/models/` | SQLAlchemy-модели |
| `app/publisher/` | Адаптеры публикации (Telegram, VK, MAX) |
| `app/routers/` | FastAPI-роутеры (REST API) |
| `app/views/` | CustomView-классы (визарды) |

## Важные соглашения (из rules/)

- **Нет Alembic.** Все миграции — аддитивные `ALTER TABLE` в функции `_migrate()` в `main.py`.
- **Все токены через `EncryptedString`.** Никогда не хранить учётные данные в поле `String`.
- **Новые формы источников:** вкладка AI Prompts отсутствует на форме создания; на форме редактирования — 2-вкладочный макет.
- **`ModelView`: параметр `icon=`** передаётся в конструктор, не как атрибут класса.
- **Starlette-admin identity:** использует `slugify_class_name` (например, `VKSource` → `v-k-source`).
- **Воркер** опрашивает pending-каналы каждые 30 секунд.
- **AIProvider:** только один может быть активен одновременно.
- **`SECRET_KEY`** обязательно менять в продакшне; дефолтное значение недопустимо.
- **Новые publisher-адаптеры** должны реализовывать контракт: `async def publish(text, source, image_paths) -> tuple[bool, str | None]`.
- **Все wizard-флоу** расширяют `CustomView` с `EditorAccessMixin` или `SuperadminOnly`.

## Запрещённые действия

- Никогда не реализовывать напрямую — только через координацию `team-lead`.
- Никогда не пропускать чтение `rules/` перед началом работы.
- Никогда не использовать Alembic или авто-миграции.
- Никогда не хранить токены и секреты в открытом виде (только `EncryptedString`).
- Никогда не коммитить изменение `SECRET_KEY` в систему контроля версий.
- Никогда не реализовывать незадокументированные API-эндпоинты.
- Никогда не пропускать шаги проектирования перед реализацией.
